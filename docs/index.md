## Quality of Life Features

ACSS is committed to solving common challenges that real web designers face daily. These unique features set ACSS apart:

- **Automatic Offsets** ‚Äì One-click fixes for overlay headers, sticky headers, sticky elements, and scroll-to-hash link spacing issues
- **String Expansion** ‚Äì Never write `var()` or `calc()` again. ACSS auto-expands `--variable` to `var(--variable)` and `1 + 1` to `calc(1 + 1)`
- **Smart Content Spacing** ‚Äì Intelligent spacing for perfect gaps between headings, paragraphs, buttons, links, and lists
- **Advanced Focus Styling** ‚Äì Choose between outline or box-shadow focus styles, customize colors, and apply with simple utility classes
- **Clickable Parent** ‚Äì Make entire card areas clickable while maintaining accessibility‚Äîaccomplished with a single utility class
- **Automatic REM Conversion** ‚Äì Use pixel values followed by "ctr" and ACSS converts them to proper REM units automatically
- **Automatic Text Balancing** ‚Äì Say goodbye to orphans with automatic heading and paragraph text balancing
- **Right-Click Contextual Menus** ‚Äì See available classes and variables with live preview directly in your page builder
- **Cheat Sheet Integration** ‚Äì Quickly reference any utility class or variable without leaving your workflow
